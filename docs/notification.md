### Notification

```js
import Notification from 'wepy-ui-ydj/Notification';
```

| 参数            | 说明                              | 类型   | 可选值                                      | 默认值                   |
| --------------- | --------------------------------- | ------ | ------------------------------------------- | ------------------------ |
| icon            | 消息通知的 icon                   | string | -                                           | ''                       |
| length          | 消息通知的最大条数,0 则表示无限制 | number | -                                           | 0                        |
| position        | 出现的位置                        | string | left-top/left-bottom/right-top/right-bottom | left-top                 |
| backgroundColor | 消息通知的背景颜色                | string |                                             | rgba(253, 105, 143, 0.9) |
| height          | 消息通知的高度                    | number |                                             | 48                       |
| fontSize        | 消息通知的字体大小                | number |                                             | 24                       |
| fontColor       | 消息通知的字体颜色                | string |                                             | #fff                     |
