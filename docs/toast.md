### Toast

```js
import Toast from 'wepy-ui-ydj/Toast'
```


| 参数       | 说明        | 类型      | 可选值                        | 默认值   |
|----------|-----------|---------|----------------------------|-------|
| show     | 控制 显隐 的变量 | boolean | -                          | false |
| content  | 显示的文本内容   | string  | -                          | 提示消息  |
| type     | 弹出框主题色    | string  | none/success/warning/error | none  |
| position | 弹出框位置     | string  | top/middle/bottom          | top   |
